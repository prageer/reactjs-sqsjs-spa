# Usage


## Install

```bash
$ npm install

```

## Build

```bash
$ npm run build

Builds Server and Client
```

## Run Dev Server

```bash
$ npm run development

http://localhost:4000/

Start the server in development mode.
Express server runs on port 5000 and development server runs on port 4000.
```

## Run Production Server

```bash
$ npm run start

```
